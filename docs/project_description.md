## Title Danish: P-PIPE: En læringsplatform til MLOps

## Title English: P-PIPE: A learning platform for MLOps

## Description Danish: 

Machine Learning er blevet et udbredt emne i samfundet, og som det stiger i poplaritet, stiger manglen på ingeniører som kan implementere det også. I takt med dette er der også opstået mange udannelser, kurser, og online værktøjer til at undervise i modellering af machine learning modeller. Det er dog sjældnere at man støder på værktøjer som underviser ligeledes i den praktisk implementation af ML, nemlig Machine Learning Operations. 

Projektet fokuserer sig på at planlægge og implementere et redskab, gerne en online platform, som kan understøtte undervisning i MLOps. Dette skal primært gøres ved at tillade brugere at opsætte scenarier som efterligner relle situationer hvor ML skal implementeres. Dette kan inkludere scenarier hvor den relevante data kommer i mindre stykker på forskellige platforme over en længere periode, frem for som én fil en enkelt gang. Dette vil så kombineres med metoder til at vurdere en brugers evne til kontinuerligt at hente og benytte sig af denne data til at træne og gentræne deres modeller. Yderligere funktionalitet kan evt. inkludere metoder til at vurdere en brugers evne til at have en responsiv model, potentielt ved at kræve intermitterende batch predictions, eller ved at kræve de pålægger et endpoint til deres model som kan lave predictions.

Endeligt, kombineres dette med begrænset undersøgelser og litteraturstudier, for at vurdere nuværende studerendes evne inden for MLOps relevante emner, samt for at overveje hvad der evt. kan gøres for at forbedre disse evner. 

## Description English:

Machine learning has become a widespread subject in society, and as it rises in popularity so does the need for engineers who can implement it. In concert with this, many education opportunities, courses, and online tools to teach modelling of machine learning, have appeared. It is, however, more rare that one encounters tools that similarily teach the practical implementation of ML, namely Machine Learning Operations.

The project focuses on planning and implementing a tool, likely an online platform, which can assist in the teaching of MLOps. This will primarily be done by allowing users to create scenarios that mimic real-world scenarios where ML needs to be implemented. This can include scenarios where the relevant data comes in smaller pieces and on different platforms over a longer period of time, as opposed to one file at a single point in time. This will then be combined with methods to judge a user's ability to continuously obtain and use this data to train and retrain their models. Additional functionality can potentially include methods to judge a user's ability to have a responsive model, potentially by requiring intermittent batch predictions, or by requiring that they attach an endpoint for their model which can make predictions.

Finally, this is combined with limited surveys and litterature studies to ascertain current student's abilities in MLOps relevant subjects, as well as consider what can potentially be done to improve these abilities.
